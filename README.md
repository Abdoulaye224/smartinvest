# memoire
Ce repo contiendra mon code source pour mon mémoire de fin de master à l'ESG Finance

le but est de créer une application intteractive pour les traders et investisseurs pour les aider dans leur quête de prises de décisions éclairées sur le marché S&P 500.

L'application met à disposition quelques indicateurs clés, l'analyse de sentiment sur les actualités liées au marché, les performances en temps réels de mes models de prédiction, la possiblité de prédire les tendances futures du marché et appliquer une stratégie de trading que j'ai développé en me basant sur le croisement des moyennes mobiles.

voici le lien vers cette web app : https://smartinvest.streamlit.app/
